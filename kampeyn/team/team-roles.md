# Kampeyn — Team Roles (v1)

## Principal / Owner (Lenox) — lenox@kampeyn.com
Owns business strategy, sales, fundraising consulting delivery standards, and product direction.

Core responsibilities:
- Revenue targets + pricing strategy
- Sales pipeline + client acquisition
- Productized service design (call time, email, events)
- Quality control on deliverables
- Product roadmap for Kampeyn ecosystem (Donor Camp, Hiring Blue, etc.)

Key KPIs:
- MRR vs target
- Active clients (3–4 target)
- Pipeline value + close rate
- Delivery SLA + client retention

## Operations (Rahele) — rahele@kampeyn.com
Owns execution and the operating system for internal workflows.

Core responsibilities:
- Intake + task assignment + follow-through
- Documentation + templates + SOP upkeep
- Vendor coordination + scheduling
- Reporting cadence (weekly)

Key KPIs:
- On-time delivery rate
- Task throughput by priority
- Client SLA adherence
