# Client Template

Copy this folder structure into `clients/<client_slug>/`.

## Suggested client_slug
Lowercase, hyphen-separated. Example: `jane-doe-for-mayor`.

## Required files
- `01-voice/voice-guide.md`
- `00-admin/scope.md`

## Optional
- `00-admin/compliance-notes.md`
- `02-briefs/YYYY-WW-brief.md`
