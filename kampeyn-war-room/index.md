# Kampeyn War Room

## What This Is
A "2nd brain" document system for Kampeyn — automatically capturing concepts, decisions, and daily journals from our work sessions.

## Folder Structure

```
kampeyn-war-room/
├── journal/          # Daily session summaries (auto-generated)
├── concepts/         # Important ideas, setup docs, how-tos
├── decisions/        # Key decisions with rationale
├── clients/          # Per-client notes (mirrors Google Drive structure)
└── index.md          # This file — auto-generated index
```
