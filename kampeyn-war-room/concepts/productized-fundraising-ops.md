# Productized Fundraising Ops — v1

Goal: productize Lenox’s fundraising consulting so delivery can be delegated and scaled.

## Package 1: Call Time Ops
**Ideal for:** candidates who need consistent donor call-time execution

**Cadence**
- Weekly call-time plan + daily/weekly targets

**Inputs required**
- Donor list access (CSV/NGP/ActBlue export)
- Candidate availability + call blocks
- Compliance constraints (state/federal)

**Outputs**
- Call sheets (prioritized)
- Follow-up tasks + notes capture
- Weekly summary (dials, connects, pledges, dollars)

**SOP skeleton**
1) Import/clean donor list
2) Segment + prioritize
3) Generate call sheets
4) Track outcomes
5) Follow-ups + thank-yous
6) Weekly report

**Pricing (placeholder)**
- $X/week or $X/month (needs Lenox input)

## Package 2: Email Fundraising Ops
**Ideal for:** campaigns running regular email fundraising without in-house copy/ops

**Cadence**
- 2–4 sends/week + rapid response

**Inputs required**
- Email platform access
- Brand voice examples
- Approval SLA

**Outputs**
- Copy drafts + subject lines + segmentation plan
- Send calendar
- Post-send reporting + iteration

**SOP skeleton**
1) Offer + goal selection
2) Draft copy variations
3) QA links/UTMs
4) Schedule + send
5) Report + learnings

**Pricing (placeholder)**
- $X/month (needs Lenox input)

## Package 3: Event Fundraising Ops
**Ideal for:** candidates doing 1–2 events/month but lacking logistics capacity

**Cadence**
- Event planning sprint per event

**Inputs required**
- Target list + host committee
- Venue + date constraints

**Outputs**
- Event plan + timeline
- Invite copy + follow-up plan
- RSVP tracking + day-of run-of-show
- Post-event thank-you + pledge chase

**SOP skeleton**
1) Define goal + ticket levels
2) Build invite list
3) Invite + follow-up cadence
4) RSVP tracking
5) Run-of-show
6) Post-event follow-ups

**Pricing (placeholder)**
- $X/event + $X/month retainer option (needs Lenox input)
