# Organize SD — Team Roles (v1)

## Executive Director (Lenox) — lenox@organizesd.org
Owns strategy, fundraising, partnerships, and overall outcomes.

Core responsibilities:
- Org strategy + goals + KPIs
- Fundraising + sponsor pipeline + donor comms
- Partnerships (campaigns, orgs, unions, tribal/community groups)
- External narrative/positioning
- Placement pipeline for graduates

Key KPIs:
- Funds raised vs goal
- Donor retention + recurring donors
- Active partner orgs + sponsor commitments
- Cohorts launched on schedule

## Training Director (Michele) — mkelly@organizesd.org
Owns curriculum quality and producing job-ready operatives.

Core responsibilities:
- Curriculum design (6–8 modules) + facilitation plan
- Trainer management + standards + run-of-show
- Practical evaluations (roleplays, field QA, coaching)
- Post-training office hours + alumni coaching

Key KPIs:
- Curriculum completion rate
- Trainee satisfaction (session ratings)
- Skills lift / practical pass rate
- Attendance rate + office-hours utilization

## Operations Director (Rahele) — rahele@organizesd.org
Owns the operating system: application → selection → logistics → records.

Core responsibilities:
- Application system + tracking + response SLA
- Scheduling interviews + maintaining pipeline dashboard
- Travel/logistics planning + vendor/venue coordination support
- Records, documentation, budgets, reporting cadence

Key KPIs:
- On-time delivery rate
- Application processing time (app → decision)
- CRM completeness + dashboard updated weekly
- Spend vs budget variance

## Campus Outreach Director (Hannah) — hdayaget@organizesd.org
Owns the campus talent pipeline statewide.

Core responsibilities:
- Recruit on campuses (USD, SDSU, etc.) + student org partnerships
- Coordinate tabling, digital boards, class drop-ins, listserv placements
- Convert leads → applications → accepted → graduated
- Maintain repeat pipeline each semester

Key KPIs:
- Campuses activated
- Events hosted + attendance
- Lead → application conversion
- Accepted → graduated retention
- Geographic spread

## Shared team norms (recommended)
- Weekly 30-min pipeline meeting during Feb 2–Feb 28
- 48-hour applicant response SLA
- Document decisions + reasons in the tracker
