# Organize SD

South Dakota-based training initiative to build Democratic grassroots capacity by training a new generation of organizers and campaign operatives.

## Key dates (2026)
- Application opens: **Mon Feb 2, 2026**
- Application closes: **Sat Feb 28, 2026**
- Training weekend (in-person): **Mar 27–29, 2026**

## What we’re building
- A clear, public-facing offer (a cohort/fellowship)
- A disciplined internal operating system for recruitment → selection → onboarding → training → placement

## Folder map
- `strategy/` — positioning, compliance guardrails, program definition
- `ops/` — checklists, owner board, pipeline SOP
- `recruiting/` — application short-form + interview script + question bank
- `comms/` — outreach templates + social plan notes
- `training/` — curriculum outline + materials list + run-of-show
- `post-cohort/` — office hours + placement pipeline + KPIs
